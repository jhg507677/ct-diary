## 기술 회고
해당 문제는 삭제된 행은 계산하면 안되기때문에 배열이 아닌 연결리스트처럼 구현했어야함
next와 prev를 저장해놓고 해당 값이 삭제될 경우에는 삭제된 값은 스택으로 저장 따로하고
해당 행의 next와 prev의 연결을 삭제해주면 됨
Class로 구현할 필요는 없고 up, down 두개의 배열이 있으면 된다.
```java
int[] up = new int[n + 2]
int[] down = new int[n + 2]
for(int i = 0 ; i < n + 2; i++){
  up[i] = i - 1;
  down[i] = i + 1;
}
```
