# ct-diary

## Algorithm & Coding Test Training Log

백엔드 개발자로서 실무 경험을 쌓아오며,  
이직을 대비해 **자료구조·알고리즘 기초부터 체계적으로 재정비**하기 위해  
코딩 테스트 풀이 기록 저장소를 운영합니다.

본 저장소는 단순 문제 풀이 모음이 아니라,  
**문제 접근 과정, 선택한 알고리즘의 근거, 개선 포인트**를 함께 정리하는 것을 목표로 합니다.

---

## 📌 목적 (Purpose)

- 코딩 테스트에서 요구되는 **문제 해결 능력의 재정립**
- 실무 위주의 개발 습관에서 벗어나 **시간·공간 복잡도 중심 사고 강화**
- 자료구조 및 알고리즘 개념을 **코드 레벨로 명확히 설명 가능**한 상태로 만들기

---

## 📚 문제 출처
- **프로그래머스 (약 90% 이상)**

- 그 외 문제는 각 디렉터리 또는 README에 출처 명시

  - 시간 / 공간 복잡도 분석
  - 실패 원인 및 개선 포인트 (있는 경우)
